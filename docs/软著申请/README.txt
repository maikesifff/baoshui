================================================================================
自报税 软件著作权登记 - 申请材料目录
================================================================================

本文件夹用于存放软著申请所需的填写内容与生成文件，便于您复制到申报系统或上传 PDF。

--------------------------------------------------------------------------------
一、表单填写内容（复制到系统对应输入框）
--------------------------------------------------------------------------------
  软著申请-填写内容.txt
  - 开发的硬件环境、运行的硬件环境、开发/运行操作系统、开发工具
  - 编程语言、源程序量、开发目的、面向领域、主要功能、技术特点
  - 各栏已按字数限制写好，可直接复制粘贴；编程语言在系统里勾选 Python、JavaScript、HTML，其他填 CSS

--------------------------------------------------------------------------------
二、程序鉴别材料（源程序前30页+后30页，上传 PDF）
--------------------------------------------------------------------------------
  源程序-前30页与后30页.html    已生成好的源程序 60 页 HTML
  源程序提交说明.txt            如何用浏览器打开上述 HTML 并打印为 PDF

  操作：用浏览器打开 .html -> 打印 -> 另存为 PDF -> 上传该 PDF

--------------------------------------------------------------------------------
三、文档鉴别材料（任选一种文档的前30页+后30页，上传 PDF）
--------------------------------------------------------------------------------
  自报税软件用户手册.md         用户手册正文（Markdown）
  文档鉴别材料说明.txt         如何将手册转为 PDF 并截取前30+后30 页

  操作：将 .md 用 Typora/Word 等转为 PDF，再取该 PDF 的前 30 页与后 30 页合并为一个 PDF 上传

--------------------------------------------------------------------------------
四、辅助脚本（可选）
--------------------------------------------------------------------------------
  生成源程序页.py              若需按最新代码重新生成「源程序-前30页与后30页.html」，
                                在项目根目录执行：python3 docs/软著申请/生成源程序页.py

--------------------------------------------------------------------------------
五、其他相关证明文件
--------------------------------------------------------------------------------
  按软著系统要求，如有营业执照等证明可另行上传。
